## HOW TO USE
- Install Python 3.7 (for Struct)
- Run 'convert.py' to create folders
- Place the .scw files in the scw folder
- Run 'convert.py' again
- Run 'combiner.py' <br>
### <b>Congratulations! Obj models created!</b>
